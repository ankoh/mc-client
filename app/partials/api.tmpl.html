Documentation

GET /system

Returns a system report that covers server-version, uptime, last-update, log-size, research-group-id and mendeley-api-status.

GET /fields

Returns all research fields that have been identified by the crawler. The crawler checks the tags that have been added to a Mendeley publication and merges small spelling variations. Thereby the original field name is reconstructed based on dash delimiters. (e.g. cyber-physical-systems becomes Cyber Physical Systems)

GET /profiles

Given the optional parameters profiles and fields, returns all Mendeley Profiles that are associated with either a profile id OR a field id. The crawler only caches those profiles that belong to the configured Mendeley Research Group. If you do not pass any of these two parameters, all profiles (instead of none) are returned. If you pass the parameter slim=true, the result will only include name, id and publication-count.

GET /publications

Given the optional parameters profiles and fields, returns all publications that are associated with a profile id AND a field id. (e.g. ?profiles=1234125,1233125&fields=12534,12535 returns all publications of the users 1234125 and 1233125 that are associated with either the field 12534 or the field 12535)
